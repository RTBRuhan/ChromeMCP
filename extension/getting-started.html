Apex Agent

Connect AI assistants to control your browser

Apex Agent lets AI tools like Cursor, Claude Desktop, or any MCP-compatible assistant control your browser. The AI can click, type, navigate, take screenshots, and debug web pages.

Follow the steps below to get started. Setup takes about 5 minutes.

1

Install Node.js

Node.js is required to run the server that connects your AI tool to the browser.

3

Configure Your AI Tool

Setting up Cursor

  1. Open Cursor
  2. Press Ctrl+Shift+J (Windows) or Cmd+Shift+J (Mac) to open settings
  3. Search for MCP
  4. Click "Edit in settings.json"
  5. Copy this config and change the path to where you extracted the folder:
{
  "mcpServers": {
    "apex-agent": {
      "command": "node",
      "args": ["C:/ApexAgent/mcp-server/index.js"]
    }
  }
}
‚ö†Ô∏è Change the path! Replace C:/ApexAgent with your actual folder path.
  1. Save the file
  2. Completely restart Cursor (close and reopen)

Setting up Claude Desktop

  1. Open the Claude config file:
    • Windows: Press Win+R, paste %APPDATA%\Claude, open claude_desktop_config.json
    • Mac: Open ~/Library/Application Support/Claude/claude_desktop_config.json
  2. Add this config (change the path to your folder):
{
  "mcpServers": {
    "apex-agent": {
      "command": "node",
      "args": ["C:/ApexAgent/mcp-server/index.js"]
    }
  }
}
‚ö†Ô∏è Change the path! Replace C:/ApexAgent with where you extracted the folder.
  1. Save and restart Claude Desktop

Generic MCP Configuration

For any MCP-compatible AI tool, add this server configuration:

{
  "apex-agent": {
    "command": "node",
    "args": ["/path/to/ApexAgent/mcp-server/index.js"]
  }
}

Refer to your AI tool's documentation for where to add MCP configurations.

4

Connect the Extension

  1. Click the Apex Agent icon in your browser toolbar
  2. Go to the MCP tab
  3. Click Connect
  4. Status should show Connected (green)
